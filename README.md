# College-Cabinet-Voting-System

A secure, transparent, and user-friendly digital platform designed to automate the election process for college student unions and cabinets. This system replaces traditional paper ballots with a digital interface, ensuring real-time results and eliminating manual counting errors.

🚀 Overview:-
In a typical college environment, electing student representatives can be a logistical challenge. The College Cabinet Voting System provides a centralized hub where administrators can manage candidates and students can cast their votes securely using their unique institutional credentials.

Key Features:-
Secure Authentication: Integration with student IDs and secure login to ensure "one student, one vote."

Candidate Profiles: Dedicated sections for candidates to display their manifestos and photos.

Real-time Analytics: Instant tabulation of results with graphical representations (Bar charts/Pie charts).

Admin Dashboard: Comprehensive control panel to add/remove candidates, manage voter lists, and monitor election status.

Audit Trail: Transparent logging to prevent tampering and ensure election integrity.

🛠️ Tech Stack:-
Frontend: HTML5, CSS3, JavaScript (Bootstrap)

Backend: PHP

Database: MySQL 

Authentication: JWT (JSON Web Tokens) or OAuth2
